# motion
- https://no822.github.io
- 드림코딩 타입스크립트 강의(https://academy.dream-coding.com/courses/typescript) 의 실전 프로젝트 스스로 구현연습한 페이지입니다.
  - 4가지 섹션 추가 기능(IMAGE, VIDEO, NOTE, TASK)
    - <입력값>: 값을 입력하지 않으면 디폴트값으로 섹션 생성
      - IMAGE: title, url
      - VIDEO: title, url(유튜브 공유 url)
      - NOTE/TASK: title, body
  - 섹션 삭제
  - 드래그 정렬 기능
